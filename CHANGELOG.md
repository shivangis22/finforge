# Changelog

## 1.0.0 - 2026-05-10

- Stabilized FinForge for v1.0 release.
- Added persistent user identity metadata and spending styles.
- Added dedicated subscription generation with stable monthly recurrence.
- Added stronger low-balance suppression and configurable overdraft handling.
- Added richer transaction metadata including recurrence, session, and balance-state fields.
- Calibrated session generation and spending frequency by behavioral profile.
- Expanded automated test coverage for recurrence, behavior, metadata, and reproducibility.
